# Desktop App Downloads

Place desktop app installer files here.

## File Naming Convention
- `HRMS Tracker Setup 1.0.0.exe` - Windows installer
- `HRMS-Tracker-1.0.0.dmg` - macOS installer (future)

## Access URL
Files will be available at: `http://YOUR_SERVER:3001/downloads/FILENAME`

Example: `http://localhost:3001/downloads/HRMS Tracker Setup 1.0.0.exe`
